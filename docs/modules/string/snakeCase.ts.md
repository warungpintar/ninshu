# string

## snakeCase

convert `string` to
[snake_case](https://en.wikipedia.org/wiki/Snake_case)

**Signature**

```ts
export declare const snakeCase: (input: string) => string
```

**Example**

```ts
import { snakeCase } from '@warungpintar/ninshu'

snakeCase('snakeCase') // output: snake_case
```
