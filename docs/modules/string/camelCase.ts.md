# string

## camelCase

convert `string` to
[camelCase](https://en.wikipedia.org/wiki/Camel_case)

**Signature**

```ts
export declare const camelCase: (input: string) => string
```

**Example**

```ts
import { camelCase } from '@warungpintar/ninshu'

camelCase('camel_case') // output: camelCase
```
